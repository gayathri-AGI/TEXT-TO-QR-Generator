TEXT-TO-QR-GENERATOR

A simple web application that allows users to generate QR codes from text or URLs. 
Built using HTML, CSS, and JavaScript.

Features:

Enter any text or URL to generate a QR code.
The QR code is displayed instantly after clicking the "Generate QR Code" button.
Includes error handling for empty input fields.

Technologies Used
Frontend: HTML, CSS, JavaScript
QR Code API: QR Code Generator API
